== Description ==

Accept Gulden in your store

Adds the feature to accept Gulden in your WooCommerce webshop. A list of features:

* Set a price in EUR and your users pay the live exchange rate in Gulden
* Fill out your Nocks.com API key, select your prefered merchant account and youâ€™re good to go

You need an Nocks-account to use this plugin and users only need a Gulden wallet to checkout at your webstore.

What is Nocks?
With Nocks you can pay at every Bitcoin merchant with Guldens and vice versa. You can use the website nocks.com to issue a cross payment.

== Installation ==

This section describes in short how to install the plugin and get it working. If you need more help, you can send an email to support@nocks.com.

1. Install the plugin via Plugins -> New plugin.
2. Activate the plugin through the 'Plugins' menu in WordPress, the name of this plugin is: Nocks Checkout for WooCommerce.
3. Under WooCommerce -> Settings -> Payment, activate the desired payment methods.
4. You can now accept Gulden payments in your store using Nocks Checkout.
